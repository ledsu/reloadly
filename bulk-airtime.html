Bulk Airtime Scheduling

API & User Portal

A quick, easy, scalable API for distributing bulk airtime across 800 Telcos worldwide. Great use when telcos have scheduled promotions, sim GSM gateways, bulk orders for customers or taking top up orders from customers via csv file.

Schedule to process airtime top ups in bulk, with an easy API or directly from our web portal.

Processing bulk airtime is a feature that allows users to upload a csv file in 3 easy steps.

Upload the csv file with the numbers and amounts you want to top up.

Select a date and time to start the bulk processing and easily setup up the schedule and Reloadly will processing the file.

Fix or remove any errors returned from Reloadly, and finally submit and schedule your bulk airtime processing file.

After submitting your file, all successful transactions will be available to on your Transaction Report in real-time.
